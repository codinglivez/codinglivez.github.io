Vending Machine Access

This mobile application is a part of a major project which includes complete end-to-end system for Smart and Cashless Vending Machine. The project report will be linked below:

  • Rating:
  • Client: AFour Technologies Pvt. Ltd.
  • Website: Project Report
  • Completed: 28 July 2020

Roughly explaining about the main project; The smart vending machine is a vending machine which will let users buy any product availible in the vending machine without physically interacting with the vending machine and the transactions will also be digital. So to access the vending machine, IoT technology is used which will help to communicate with the machine.

The mobile application allows the user to know what is present in the vending machine he/she is standing in front of and choose which product the user has to buy. Then the application is connected to Payment API's which help to complete the payment process and as soon as the API sends the Payment Completion acknowledgement to the system, the product demanded by the user is dispatched from the machine and the user gets intended product without even touching the machine.
