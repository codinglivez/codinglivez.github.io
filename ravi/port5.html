College Website

This is a simple Website based on Bootstrap Framework for general introduction and the formal content which aquires with the information regarding the college.

  • Rating:
  • Client: Government Polytechnic Pen
  • Project Link: gppen.ac.in
  • Completed: 14 February 2019

This site is based on HTML, CSS and little bit of JavaScript for minimal animation in some sections of the design. The site is built according to the instructions and guidelines provided by the cleint and it took plenty of effort to collect the huge amount of data from the client in small parts and then enter it onto the design.

This site took the longest time to be built as many formal procedures were implemented while building the site. The site did not end up as expected by the team because of artistic restrictions from the client end. The site is responsive and supports huge range of devices The data on the site is regularly updated according to the instructions recieved from the institute. This is one of the sites which I constructed when I had very less experience.
