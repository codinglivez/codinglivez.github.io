Todo Application

This Web Application is an attempt to make an simple-to-use To-do list which will help to just jot down anything along time which can be referred later.

  • Rating:
  • Client: Personal Project
  • Project Link: codinglivez.github.io/todo/
  • Completed: 26 March 2020

This Web Application is based on HTML 5, CSS 3 and JavaScript to handle the list items on the scripting end and to store them as expected. The App introduces you to a text input box where user can enter the task to be completed and the task then appears in the list just below the box. The list then shows you 2 options namely Done and Delete which perform the task named

The done button adds a strike through the word making it showcase that the task has been completed and need not to worry about it whereas the delete button just removes the task completely from the list with a sweet drop animation. The list also has a small categorization which makes you to switch between completed and incomplete tasks. This will make everything more sorted and you can focus on only what you need to perform ahead.
