PHP for beginners: Building your first simple CMS

By Tanvee Gawand| 18 June | 2 Comments | PHP

Now a days every up-to-date website using some form of content management system (CMS). CMS provide no of free options to power a website( Wordpress, Drupal, etc.), It doesn’t hurt to peek under the hood and get feel for how these system work.

As we are back-end developer, we’ll be create a simple PHP class that will:

  • Create a database
  • Connect to a database
  • Display a form data in the database
  • Save the form data in the database
  • Display the saved data from the database

This class is intended to create interaction between PHP and MySQL together, and to show basic of CMS.

Building the Class

Our first step is to simple layout the class in file named 'simplecms.php'. While using PHP use object oriented approach because it makes for cleaner code in large project, and, in my opinion, it’s just simple to understand.
